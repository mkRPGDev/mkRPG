# Utilisation:
(Se placer dans src)

### Serveur
Generic game server.

usage: server.py [-h] [-p PATH]

### Client
Generic game client.

usage: client.py [-h] [-p PATH] [-c | -g | -n] [-d]

### Lancement
lancer le serveur puis deux clients avec python3 client.py -g
