# Utilisation:
(Se placer dans src)

### Serveur
Generic game server.

usage: server.py [-h] [-p PATH]

### Client
Generic game client.

usage: client.py [-h] [-p PATH] [-c | -g | -n] [-d]


Un sytème de plugins est disponible (et en amélioration) cf plugins/ ...


### Lancement
lancer le serveur puis le nombre requis de clients avec python3 client.py -g
